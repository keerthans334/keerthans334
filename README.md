# üí´ About Me:
I am currently working on multiple mini projects focused on AI, Machine Learning, and cloud-based systems, with a strong interest in building technology that creates real-world impact.

One of my key projects is FarmAura, a Smart Crop Advisory System designed for small and marginal farmers. This project was showcased at the Smart India Hackathon (National Level), where I served as the Team Leader of Group 6.
FarmAura is built on a Machine Learning model trained using:
Soil micro- and macro-nutrient data
Region-specific climate conditions
Land nutrition and fertility parameters
Based on these inputs, the system provides crop and fertilizer advisory recommendations to farmers in their native language, using an IVR-based interface with offline capabilities, making it accessible even in low-connectivity rural areas.
